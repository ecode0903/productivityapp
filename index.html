FEATURES

Check out what you can do with this app!

Action Cards

Create Action Cards to note things you do not want to forget.

Flexible Use

Type in “Post status report every second Friday” and Actions takes care of the scheduling, reminders and notifications.*

Drag and Drop

Drag and drop your Action Cards into custom Lists to organise them intuitively

Color Code

Color code each list to further personalize your productivity system

Schedule

Use the Schedule to see all of the Actions coming up

Swipe

Swipe on Action Cards to complete them, delete them and set reminders.
